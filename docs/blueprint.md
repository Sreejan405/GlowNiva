# **App Name**: GlowGenius

## Core Features:

- AI Product Recommendation: Recommends the most suitable facial mask from the catalog using AI to analyze user preferences and skin types, and tool to incorporate data from reviews.
- Product Browsing: Browse and filter available facial masks, including details like ingredients, benefits, and skin types.
- Product Detail Pages: View detailed information for each mask, including high-quality images, descriptions, ingredients, and customer reviews.
- Customer Reviews: Users can view product reviews.
- Contact Form: Implement a contact form for user inquiries and feedback.
- About Us Page: Create an 'About Us' page to share the brand's story, values, and mission.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke a sense of calm and luxury.
- Background color: Light gray (#F5F5F5), almost white, to give an airy, clean feel to the overall site.
- Accent color: Muted rose (#D8BFD8) for CTAs and interactive elements, providing a delicate contrast.
- Headline font: 'Playfair' serif for headings, lending elegance and readability to the site. Note: currently only Google Fonts are supported.
- Body font: 'Inter' sans-serif for body text, balancing a modern and clean aesthetic for clear content presentation. Note: currently only Google Fonts are supported.
- Use minimalist line icons to represent product features, benefits, and other relevant information.
- Employ a clean and spacious layout, with a focus on high-quality product imagery.
